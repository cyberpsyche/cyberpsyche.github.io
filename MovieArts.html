Movie 看电影

Run Forrest, RUN ... ---------Brad Pitt, "Fight Club"

看电影是件有趣的事，我乐此不疲，我常会注意电影中甚至毫不相干的路人，想像他／她们永远被定格 背后的人生是什么样的。《罗拉快跑》里就有把这种想像影像化的手法，这样的想像更让我觉得每个人 所能亲眼见到的世界真的是冰山一角。毫不怀疑, 电影在有形地延伸人的视线，无形地延长人的生命。

Java Notes

compile class

javac Helloworld.java

make jar file

1, jar cvf hello.jar ~/path/*.class

2, unzip hello.jar

3, Append a line "Main-Class: HelloWorld" to META-INF/MANIFEST.MF, then

jar umf META-INF/MANIFEST.MF hello.jar

4, Run the jar file

java -jar hello.jar arg1 arg2 ...

在eclipse中导出jar文件方法： File -> Export...

  1. 简化为脚本

制作一个hello.sh的脚本，可接受参数，内容为： 

#!/bin/bash
java -jar ~/bin/hello.jar "$@"

Tips

查看java版本

java -version

引用jar中的文件

java -cp path/to/filename.jar ...

解开jar包

jar xvf path/to/filename.jar

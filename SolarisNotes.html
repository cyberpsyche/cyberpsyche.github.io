Solaris 包管理

Solaris下的软件包一般可以在 http://www.sunfreeware.com/ 下载到，包名形如gcc-3.4.6-sol10-sparc-local.gz。

在安装之前，必须对软件进行解压缩。使用以下命令

gunzip gcc-3.4.6-sol10-sparc-local.gz

解压缩后，会得到一个gcc-3.4.6-sol10-sparc-local的文件。可以用命令pkginfo来检查包的信息

pkginfo -d gcc-3.4.6-sol10-sparc-local

得到的信息

application SMCgcc gcc

application 安装的种类 SMCgcc 安装包的名称 gcc 软件的名称

然后就可以进行安装了。

安装软件包的命令格式

pkgadd [-d [ devicepathname ] ] pkg_name

可以使用pkgchk命令来检查是否安装成功。

命令格式

pkgchk [ option ] [-p path ...] [pkg_name]
